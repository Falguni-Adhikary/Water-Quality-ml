# Water Quality ML Project

This project uses Machine Learning to predict water potability based on chemical parameters such as pH, sulfate, turbidity, and conductivity.

## Dataset
Water Potability dataset (Kaggle)

## Model Used
Random Forest Classifier

## Workflow
- Data preprocessing
- Missing value handling
- Train-test split
- Model training
- Model evaluation

## Accuracy
~67%
